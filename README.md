# 📊 Retail Sales Analysis Dashboard - Power BI Project

## 📋 Project Overview

A comprehensive Retail Sales Analysis Dashboard built using Power BI, Power Query, and DAX. This project demonstrates end-to-end data analysis, including data cleaning, transformation, KPI creation, dashboard development, business insights generation, and strategic recommendations.

The objective of this project is to analyze retail sales performance, identify profitability challenges, and support business decision-making through interactive visualizations.

---

## ⭐ Key Features

### 1. 🧹 Data Cleaning & Transformation

Performed comprehensive data cleaning using Power Query:

* Validated Order Date and Ship Date data types
* Standardized text formatting across columns
* Verified Order Date is before or equal to Ship Date
* Checked Row ID uniqueness
* Validated Category and Sub-Category relationships
* Verified Sales and Quantity values
* Validated Discount values
* Checked Profit values for potential outliers
* Removed duplicate records
* Verified no missing values

### 2. 📊 KPI Development

Created key business metrics using DAX:

* Total Sales
* Total Profit
* Total Quantity
* Profit Margin

### 3. 📈 Interactive Dashboard

Developed an interactive Power BI dashboard featuring:

* Sales by Region
* Profit by Category
* Sales by Segment
* Sales Trend by Year
* Customer Performance Analysis
* Interactive Slicers

  * Region
  * Segment
  * Category
  * Ship Mode

## 📊 Business Insights

### 🌍 Regional Performance

* West region generated the highest sales and profit.
* East region delivered strong profitability with healthy margins.
* Central region generated the lowest profit margin despite relatively high sales.

### 🚚 Ship Mode Performance

* Standard Class generated the highest sales and profit.
* Same Day shipping generated the lowest sales and profit.

### 🛒 Category Performance

* Technology generated the highest profit.
* Office Supplies delivered strong profitability.
* Furniture generated the lowest profit margin despite strong sales volume.

### 👥 Customer Segment Analysis

* Consumer segment generated the highest sales and profit.
* Home Office segment generated the lowest sales and profit.

### 🔍 Root Cause Analysis

A detailed drill-down analysis revealed:

* Furniture sales within the Consumer segment in the Central region generated negative profits.
* This indicates pricing, discounting, or operational inefficiencies within this specific business area.

---

## 💡 Recommendations

### Furniture Category Improvement

* Investigate low-profit Furniture products.
* Review pricing and discount strategies.
* Optimize product mix within the Furniture category.

### Central Region Profitability

* Analyze cost structure and pricing practices.
* Identify loss-making products and transactions.
* Review regional sales strategies.

### Business Growth Opportunities

* Continue leveraging successful practices used in the West region.
* Expand focus on high-performing Technology products.
* Maintain strong engagement with Consumer segment customers.
